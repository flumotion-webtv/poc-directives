AngularJS directives for WebTV's

Directives are markers on a DOM element (such as an attribute, element name, comment or CSS class) that tell AngularJS's HTML compiler to attach a specified behavior to that DOM element or even transform the DOM element and its children.
More info here.

Benefits:

  • Easy way to embed code with no iFrames required.
  • Enable SEO thanks to AJAX crawling.
  • Communication between different directives.
  • Markup externalized as templates, so you can customize directive's markup at will.
  • Documentation available using ngDocs: fmt-player (work in progress).

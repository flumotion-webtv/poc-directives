AngularJS directives for OVP's

  • Enable SEO with AJAX crawling.
  • Communication between directives on a transparent way.
  • Manage different players with the same wrapper.
  • Different devices and formats including MPEG-DASH support.
  • No iFrames required.
  • No code changes on client's site to use a new release.
  • Markup externalized as templates, so you can customize directive's markup at will.
  • Documentation with ngDocs. (work in progress).

Code to embed a player, addthis and playlist using default configurations:

<!-- Player wrapper -->
<div fmt-player></div>
<!-- Social sharing -->
<div fmt-addthis></div>
<!-- Playlist synchronized with player and social sharing -->
<div fmt-playlist content="http://flumotion-webtv.github.io/poc-directives/fixtures/playlist.json"></div>

<!-- Initialize AngularJS and directives bootstrap -->
<script src="//cdnjs.cloudflare.com/ajax/libs/jquery/2.1.0/jquery.min.js"></script>
<script src="//cdnjs.cloudflare.com/ajax/libs/angular.js/1.2.15/angular.min.js"></script>
<script src="http://flumotion-webtv.github.io/poc-directives/fmt-tpls.min.js"></script>
